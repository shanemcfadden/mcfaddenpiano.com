# mcfaddenpiano.com

[mcfaddenpiano.com](https://mcfaddenpiano.com) is a static website for my freelance music work built using Gatsby. It contains a bio, photos, and videos.

[![mcfaddenpiano.com screenshot](screenshot.jpg)](https://mcfaddenpiano.com)

## Tools used

- React
- TypeScript
- Gatsby (a React framework for static websites)
- Tailwind CSS
- Netlify CDN

## Running the project locally

Requires Node.js and pnpm.

1. Clone the repo
1. Run `pnpm install` to install dependencies
1. Run `pnpm dev` to start a local development server
